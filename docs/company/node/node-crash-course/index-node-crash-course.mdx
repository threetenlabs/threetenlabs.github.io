This is a complete online course for Node.js developers wanting to learn about Redis,
the ioredis client and Redis Stack.

In this course, you'll learn about using Redis with Node.js through a blend of video and
text-based training. You can also get hands-on with some optional workshop exercises
where you'll add new functionality to an existing Node.js application.
